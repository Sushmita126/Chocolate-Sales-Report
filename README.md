# Chocolate-Sales-Report
This project showcases a dynamic Power BI report built from a global chocolate sales dataset. It includes interactive visualizations, key performance indicators, and regional insights to help understand product performance, sales trends, and team contributions.

Project Overview: Description of the sales report and data sources.
Key Metrics:

Total Sales Amount
Total Customers
Total Boxes Sold


Top Insights:

Top 5 Products by Sales
Top 5 Sales People
Sales by Region


Visualizations Used: Bar charts, pie charts/maps, KPIs, and time series.
